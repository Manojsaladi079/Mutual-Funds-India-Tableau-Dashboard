# Indian Mutual Funds Analysis Dashboard (Tableau)

## Overview
Built an interactive Tableau dashboard analysing 814 Indian 
mutual fund schemes — covering fund size, expense ratio 
across schemes, and 1-year return performance.

## Live Dashboard
https://public.tableau.com/app/profile/saladi.manoj/viz/Mutual-Funds-India-Dashboard/Dashboard1

## Visuals
- Top 10 Funds by Size (Fund Size Cr)
- Expense Ratio by Fund (all schemes)
- Top 10 Funds by 1-Year Return

## Dataset
Source: Kaggle — Mutual Funds India - Detailed
Columns: Scheme Name, Min Sip, Min Lumpsum, Expense Ratio, 
Fund Size Cr, Fund Age Yr, Returns 1Yr/3Yr/5Yr

Tableau Public
